# RF_Shield_Beaglebone_Black

PCB de doble capa diseñado para usarse como shield de la Beaglebone Black, como interfaz de un sistema de comunicaciones RF. Se utilizó Proteus 7 (ISIS/ARES) para el diseño del esquemático y PCB. El prototipo se fabricó con una fresa CNC.
